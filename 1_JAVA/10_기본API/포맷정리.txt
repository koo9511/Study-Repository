[format : 출력형식]

 - 값 출력시 format 을 적용해서 출력할 수도 있음
	System.out.printf("출력문장 + 출력할 값에 대한 포맷 + \문자", 
			포맷에 적용할 값);
 - 포매터(formatter)는 반드시 앞에 % 붙임
	: java.util.Formatter 에서 제공됨
 - 값에 따른 포맷(format)
	문자하나(char)	%c
	문자열(String)	%s
	정수(byte/short/int/long)
		10진수(decimal)	%d
		8진수(octal)	%o
		16진수(hex)	%x
	실수(float/double)
		지수형(과학용표기식)	%e
		부동소수점형(비과학용표기식)	%f : 소수점아래 6째까지 출력	
		%g : 기록된 값만 출력
		%A : 16진수 처리된 실수값 출력
	날짜	년도	%y
		월	%M
		일	%d
	시간	시	%h
		분	%mi
		초	%s

 - %와 포맷문자 사이에 숫자 사용 가능함
	: 출력시 사용될 화면상의 칸수를 의미함
	  양수는 정해진 칸의 오른쪽 정렬
	  음수는 정해진 칸의 왼쪽 정렬됨
	예>
	%10d : 정수값을 십진수로 화면에 10칸 차지하고 오른쪽정렬 출력하라.
	%-10d : 정수값을 십진수로 화면에 10칸 차지하고 왼쪽정렬 출력하라.
	%10.2f : 실수값을 화면에 10칸 차지하고 소숫점 아래 둘째자리까지
		오른쪽 정렬 출력하라.
