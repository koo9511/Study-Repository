[this 레퍼런스]

 - this 레퍼런스
	: 클래스 안의 생성자와 static 메소드를 제외한 모든 setter, getter, 
	일반 메소드 안에는 this라는 레퍼런스 변수가 존재함
	클래스 안 메소드 {} 안에서 사용되는 클래스에 소속된 필드와 메소드사용
	앞에 자동으로 붙여짐
	생략할 수 있음

	생성자 또는 메소드 {
		[this.]필드명
		[this.]메소드명([값,....]);
	}

	생성자 또는 메소드를 실행시킬 때 메소드 앞의 레퍼런스가 가진 주소를
	자동으로 전달받도록 되어 있음.

	new 생성자(초기값, .....);  ----> 생성자 안의 this가 할당된 객체의
				주소를 자동으로 전달받음
	[자료형 변수명 = ]레퍼런스.메소드명([전달값,.....]);
		-----> 앞의 레퍼런스가 가진 주소를 메소드 안의 this가
			자동으로 전달받음
