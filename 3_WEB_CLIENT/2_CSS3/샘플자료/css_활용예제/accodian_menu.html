HTML5

HTML5는 HTML의 차기 주요 제안 버전으로 월드 와이드 웹의 핵심 마크업 언어이다. 2004년 6월 Web Hypertext Application Technology Working Group(WHATWG)에서 웹 애플리케이션 1.0이라는 이름으로 세부 명세 작업을 시작하였다. 2013년 4월 현재에도 개발 중이다. HTML5는 HTML 4.01, XHTML 1.0, DOM Level 2 HTML에 대한 차기 표준 제안이다. 최신 멀티미디어 콘텐츠를 브라우저에서 쉽고 용이하게 볼 수 있게하는 것을 목적으로 한다.

CSS3

css는 웹페이지의 스타일과 레이아웃등을 제어하는 언어 css3는 css의 제일 최신 표준이라고 생각하시면 됩니다. W3C라는 단체에서 아직도 꾸준히 개발중 대부분 브라우저에서 구현이 되는데 익스플로러는 css3는 모듈로 분활 말 그대로 텍스트에다가 포토샵으로 말고 코드로 효과를 주거나 이미지에 애니메이션을 준다거나 한다 예전에 자바스크립트로 어렵게 구현했던 것을 깔끔하고 손쉽고 배우기 쉬운 코드로 구현할 수 있습니다.

API

Opensocial Javascript API and Opensocial Lightweight Javascript API The Naver Social App Platform provides opensocial Javascript API (hereinafter, Opensocial API) by default, and adds “opensocial-0.9” in the feature property of the element. In opensocial specification v0.9, opensocial lightweight Javascript API (hereinafter, opensocial lightweight API), which is a set of new APIs, is newly included. Opensocial lightweight API provides all the main functions of existing opensocial APIs, but much simpler to use. Opensocial lightweight API adds “osapi” library.
